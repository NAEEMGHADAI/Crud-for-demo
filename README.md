## Tech Stack

**Client:** HTML, CSS, JavaScript

**Server:** NodeJS, ExpressJS, MongoDB

## Run Locally

Clone the project

```bash
  git clone https://github.com/NAEEMGHADAI/Crud-for-demo.git
```

Go to the project directory

```bash
  cd Crud-for-demo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
